# .github

Organization-wide GitHub configuration and community health files for [ReadAI Team](https://github.com/readai-team).

## Overview

This repository contains default community health files and templates that apply across all repositories in the **readai-team** organization.

## Contents

| File | Description |
|------|-------------|
| `profile/README.md` | Organization profile displayed on the GitHub org page |
| `CONTRIBUTING.md` | Default contribution guidelines for all repositories |
| `CODE_OF_CONDUCT.md` | Code of conduct for community interactions |
| `SECURITY.md` | Security vulnerability reporting policy |
| `FUNDING.yml` | Sponsorship and funding configuration |
| `ISSUE_TEMPLATE/` | Default issue templates (bug report, feature request) |
| `PULL_REQUEST_TEMPLATE.md` | Default pull request template |

## How It Works

GitHub automatically uses files from this `.github` repository as defaults for any repository in the organization that doesn't have its own version of that file. This includes:

- Issue and PR templates
- Contributing guidelines
- Code of conduct
- Security policy
- Funding information
